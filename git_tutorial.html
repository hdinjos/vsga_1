Ini tutorial GIT

membuat direktori lokal

git init, ini inisialisasi

git --global user.name "kadek", membuat user di working direktori

git --global user.email "kdek@hotmail.com", membuat email di working direktori

menambahkan file ke working direktori, contoh: tutorial_git.html

git add nama_file, jika banyak menggunakan "git add ." membuat file ke staging index

git commit -m "keterangan commit", boleh tidak ditambahkan -m jika keterangannya panjang. nanti text editor akan muncul untuk memasukkan keterangannya

meremot dan mengepush ke repositori github

git remote add github htpps://nama_situs, untuk github bisa di isi dengan nama keinginan. ada dua cara menggunakan https atau ssh. tutorial ini menggunakan https

git push github master, ini mengupload ke github dengan branch utama yaitu master

mengepull atau fetch dari github ke lokal

persiapkan direktori lokal

git init, ini inisialisasi

git --global user.name "kadek", membuat user di working direktori

git --global user.email "kdek@hotmail.com", membuat email di working direktori

git remote add github htpps://nama_situs, untuk github bisa di isi dengan nama keinginan. ada dua cara menggunakan https atau ssh. tutorial ini menggunakan https

git pull github master, atau git fetch github master. perbedaannya pull digunakan jika file lokal tidak memiliki commit baru, kalau fetch digunakan jika file lokal memiliki commit terbaru
